Privacy

fullmontis.com is a static HTML webpage and as such it does not collect any information for advertising, logging, commenting or any other data collection related activities. It does not store cookies and it does not use third party software that may use them.

The website is hosted using GitHub Pages, and as the data processors for this website they may hold onto certain information for legal and security reasons. From their Privacy Policy:

Please note that GitHub may collect User Personal Information from visitors to your GitHub Pages website, including logs of visitor IP addresses, to comply with legal obligations, and to maintain the security and integrity of the Website and the Service.

If for any reason you want to act on any personal data collected on fullmontis.com, please contact me for any information and action regarding it.
